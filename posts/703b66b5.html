# 写在前面

除此之外，如果有哪里写的不好或者出现错误的情况，或者有哪里不懂的地方，请在评论区或者 QQ 联系我！作者的 QQ 是 2515579692

# 简介

  • C++ 是一门计算机高级程序设计语言，是 C 语言的一个拓展，如果有 C 语言基础的同学，在本篇你可以看到很多熟悉的影子...

  • C++ 还是一门面向对象的编程语言，至于这个面向对象是什么，我们会在下一个系列介绍。总而言之，面向对象是 C++ 的重要特性之一。

  • C++ 还可以进行泛型编程，但作者水平有限，还未完全掌握这方面的内容，所以目前并不会有这个方向的教程

# 数据类型

在 C++ 中，数据的类型十分重要，如果使用了错误的类型，可能会得到意想不到的结果，甚至可能无法通过编译...

下面我们就来介绍 C++ 中最基本的一些数据类型

  • int： 整型，用来表示整数，能表示 32 位整数
  • long long: 长整形，可以用来表示更大的整数，能表示 64 位整数
  • float: 浮点类型，可以用来表示小数
  • double: 双精度浮点类型，可以表示更精确的小数
  • char: 字符类型，用于表示一个字符
  • bool: 布尔类型，只有两种可能，true 或者 false，可以想象成一个灯只有开和关两种可能

了解了这些数据类型之后，我们就可以来了解变量，常量，函数等概念了...
